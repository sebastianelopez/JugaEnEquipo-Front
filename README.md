# Juga En Equipo

This is the front of the "Juga en Equipo" project, which will allow players from different platforms to connect, interact, create teams to compete in an amateur and professional way and why not, be contacted by professional teams.

## Available Scripts

First, run the development server:

```bash
yarn dev
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
